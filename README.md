# llm-evaluation-aiaudit
This repository contains the creation and evaluation of a language model that answers questions about the latest trends in third-party AI audits based on a 2023 report by the Regulatory Governance Initiative. The evaluation was conducted with the Giskard library, and focuses on "hallucinations".

The file giskard-technical-exercise is the html version of the Python Notebook where I conducted the work. To view the ouputs of the notebook, you can download the html file and open it with a browser of your choice.
